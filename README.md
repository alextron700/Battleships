# My C++ Battleships Project.
A digital recreation of a classic board-game battleships which runs in a terminal. Built as a learning project as I learn C++.

## Requirements
This is built in windows using microsoft visual C++. 

## Features
- A Confuigurable computer opponent with 8 difficulty settings.
- Uses ANSI escape codes for coloe
- You can save your boards with a password-based system
- Uses ascii art to represent the board.
